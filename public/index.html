Image Search Abstraction Layer API


This API is a Free Code Camp - Back End Development Certification project.

This API can be used for image search requests and makes use of the Bing Search API.
Requests can be made by passing a search query to /api/imagesearch/<search_string_goes_here>

Results can be paginated through using an offset specifying the page number like so /api/imagesearch/<search_string_goes_here>?offset=<page_number_goes_here>

Additionally recent search queries can be viewed at /api/latest/imagesearch

A JSON object is sent in each response containing either the search results or recent search terms depending on which endpoint was used

Example Usage

https://image-search-nburazor.herokuapp.com/api/imagesearch/rick sanchez peace among worlds

https://image-search-nburazor.herokuapp.com/api/imagesearch/rick sanchez peace among worlds?offset=3

https://image-search-nburazor.herokuapp.com/api/latest/imagesearch
